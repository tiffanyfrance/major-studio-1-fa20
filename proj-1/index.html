USING SMITHSONIAN INSTITUTION OPEN API

An Analysis of Topic Tagging by Gender

This project uses the open API from the Smithsonian Institute. The data is being retrieved from four separate units: the Smithsonian National Museum of American History, the American Art Museum, the Portrait Gallery, and the National Museum of African American History and Culture. The intent is to analyze gender difference by using the three most popular names, by gender, for the 1800s - late 1900s (the time period reflected in the collection). These names were gathered from the Social Security Administration website as well as several smaller websites (for data before SSA was active).
